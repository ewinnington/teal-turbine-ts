# Typescript common tasks

Install npm and node.

Install the typescript compiler into npm

```
npm install -g typescript
```

# Redis 

Setup the connection using node-redis.

```

npm install redis
npm i --save-dev @types/redis

```

Compile and run with 

```
tsc
node dist/main.js
```
